# Как обойти блокировку Авито (все прокси в блоке)

Если все твои прокси показывают «блок (доступ ограничен по IP)», есть три рабочих варианта.

---

## 1. ZenRows (рекомендуется)

**Суть:** Запросы к Авито идут через сервера ZenRows, а не с твоего IP. Свой прокси не нужен.

1. Зарегистрируйся: https://www.zenrows.com  
2. Возьми API-ключ в кабинете.  
3. В `.env` добавь строку (подставь свой ключ):
   ```env
   ZENROWS_API_KEY=твой_ключ_с_zenrows
   ```
4. Прокси в `.env` можно **удалить или закомментировать** (`# AVITO_PROXY=...`).  
5. Перезапусти бота. Парсер будет сначала ходить через ZenRows; Авито блокировать не будет.

Тарифы: есть бесплатный лимит (около 1000 запросов), дальше платная подписка.

---

## 2. ScrapingBee (альтернатива)

То же самое, но другой сервис: https://www.scrapingbee.com  

В `.env`:
```env
SCRAPINGBEE_API_KEY=твой_ключ
```
Прокси для Авито не нужен.

---

## 3. Без платных API: мобильный интернет

Если не хочешь платить за ZenRows/ScrapingBee:

1. Раздай интернет с телефона (мобильные данные).  
2. В `.env` **убери или закомментируй** строку с прокси:
   ```env
   # AVITO_PROXY=...
   ```
3. Подключи комп к раздаче с телефона и запусти бота.  
4. В боте нажми «Проверить прокси» — должно быть «ок, объявления видны».

Минус: парсить получится только когда комп в интернете через телефон.

---

## Итог

| Вариант              | Сложность | Деньги      |
|----------------------|-----------|-------------|
| ZenRows в .env       | Минимум   | Есть бесплатный лимит |
| ScrapingBee в .env   | Минимум   | Аналогично  |
| Мобильный интернет   | Без оплаты | Нет        |

**Рекомендация:** завести ZenRows, прописать `ZENROWS_API_KEY` в `.env` и не использовать свой прокси для Авито.
